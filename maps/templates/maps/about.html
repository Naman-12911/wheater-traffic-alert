Directions

Maps provides directions for walking, biking, driving, or taking public transportation.

Traffic Updates

Maps includes live traffic updates, so you can avoid traffic jams and get to your destination faster. See real-time traffic conditions for your route on the map.

Weather Updates

This map displays the latest and most realistic view of Earth from space, as taken from weather satellites. It provides a detailed view of clouds, weather

How to use?

The app will have a Map. Your current location will be point A and you will select Destination as point B.
The map will create a route from A to B. The user will get notified when there is traffic in route or weather conditions change at any part of the route.
Please let me know if you want to change the implementation or add something to this.

Technology

Django

Django is a free and open-source, Python-based web framework that follows the model–template–views architectural pattern.

Map (OpenLayers)

OpenLayer is completely free, Open Source JavaScript. OpenLayers makes it easy to put a dynamic map in any web page. It can display map tiles, vector data and markers loaded from any source. OpenLayers has been developed to further the use of geographic information of all kinds.

Search Locations (Photon komoot)

Photon is an open source geocoder built for OpenStreetMap data. It is based on elasticsearch - an efficient, powerful and highly scalable search platform. It provides search result with coordinates which can be used with most of the map and aswell we openlayers.

TOM TOM (Traffic API)

A highly detailed map that's trusted by the world's biggest companies to provide guidance and coverage to users in over 200 countries and territories.

Flow Segment Data

This service provides information about the speeds and travel times of the road fragment closest to the given coordinates. It is designed to work alongside the Flow Tiles to support clickable flow data visualizations. With this API, the client side can connect any place in the map with flow data on the closest road and present it to the user.
